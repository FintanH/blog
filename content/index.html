What

Our team is committed to exploiting the benefits of Functional Programming, and assisting others in doing same.

If Functional Programming is so much better, why isn't everyone using it yet? Because you haven't knocked on our door yet.

How

We provide courses in Functional Programming from introductory through to applied, with more in development.

If there are gaps in the ecosystem, or things that would benefit the community at large, then we will develop solutions and release them. All open source, of course.

Focus

Some of the projects we're concentrating on at the moment:

  • Programming language tools to deal with the problems associated with legacy programming languages.
  • Numerical computing and associated tools.
  • Reflex, as an implementation of Functional Reactive Programming.
  • Conducting hands-on Functional Programming courses, targetted toward industry programmers.
